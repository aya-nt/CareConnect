# Smart Home Safety System

A smart home safety solution using **ESP32 sensors**, **AI models**, and a **React Native mobile app**.  
The system detects falls, loud noises, and abnormal events, then sends alerts to family members through a Python backend with an SQL database.

## 🚀 Features
- Fall detection (AI + camera)
- Loud sound / scream detection
- Real-time sensor monitoring (ESP32)
- Fast notifications to family members
- Next.js mobile app
- Python backend with SQL database

## 🏗 Technologies Used
- **React Native** (mobile app)
- **Python FastAPI** (backend)
- **SQL** (database)
- **ESP32 + Arduino IDE** (hardware)
- **AI detection models** (falls & sound)

## 📡 System Architecture
ESP32 → Python Backend → SQL Database → React Native App
